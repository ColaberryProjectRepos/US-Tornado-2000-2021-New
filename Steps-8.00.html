Download and Load CSV File to Power BI Desktop 8



Dataset Links: 
Download dataset from Kaggle and save it under your desktop.
https://www.kaggle.com/datasets/danbraswell/us-tornado-dataset-1950-2021

us_tornado_dataset_1950_2021 (1).csv4.04 MB 



  1.      Open Power BI Desktop
  2.       Click on Get data button.
  3.      Select  Text/CSV and click on connect.
  4.       Select the us_tornado_dataset_1950_2021.csv file and click on Open.
     A preview of the data will be displayed, click on Transform
