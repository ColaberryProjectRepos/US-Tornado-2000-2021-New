Adding the Refresh Date 18

  •  Go to the Visualizations pane and select the Card Visual.
  •  From Refresh Date table, drag DateTime into the Fields section.
  •  Make the following changes: 
  • General: Properties, Size, 
    •  Height=88, 
    •  Width=165
    •  Position 
    •   Horizontal=1077
    •   Vertical=2
    •   Title, Text=Refresh Date 
    • Effects-Background-On, Color=#FFFFFF,Transparency=0%
    • Visual Border-off, Shadow-Off
  • Visual:    
  •     Callout value: Font=Arial,size-10   Font color=#070707,
  •     Display units: Auto, Value Decimal places-Auto
  •      Category label=Off
