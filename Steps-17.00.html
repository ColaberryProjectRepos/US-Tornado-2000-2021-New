Adding Magnitude (EF Scale) Slicer and Year Slicer 17

  •  To Create the Magnitude (EF Scale) slicer, from the visualizations pane select Slicer
  •    Under the Fields Pane from  US_Tornado_2000_2021 table drag Magnitude (EF Scale) column to the fields value box
       Fill up figures provided below for the necessary fields
General: Properties, Size
