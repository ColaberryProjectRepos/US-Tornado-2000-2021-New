Creating the Measures 14
Measures (Dax Formulas)
From the "Home" tab in ribbon, click "Enter Data" and just rename the blank table to "MeasuresForFuel"  and click OK. Click on Measures table in the fields section and then right click and choose 'new measure' and enter 1 measure formula from below and click tick next to it to create that measure. Right click on the measure table to create another measure and repeat the same steps until all the below measures are created in "MeasuresForTornado" table.
 Write  the following DAX Measures on the space provide one at a time and choose the necessary format and decimal for each value (measure).

1.Count = CALCULATE(COUNTROWS('US_Tornado_2000_2021'))
2.Fatalities = SUM(US_Tornado_2000_2021[Fatalities])
3.Injuries = SUM(US_Tornado_2000_2021[Injuries])
4.Length = AVERAGE('US_Tornado_2000_2021'[Length])
5.Width = AVERAGE(US_Tornado_2000_2021[Width])
6.YoY Count = 
VAR SelectedYears = VALUES('Calendar'[Date].[Year])
RETURN
    AVERAGEX (
        SelectedYears,
        VAR CurrentYearCount = CALCULATE ( COUNTROWS ( 'US_Tornado_2000_2021' ), 'Calendar'[Date].[Year] = EARLIER ( 'Calendar'[Date].[Year] ) )
        VAR PreviousYearCount = CALCULATE ( COUNTROWS ( 'US_Tornado_2000_2021' ), 'Calendar'[Date].[Year] = EARLIER ( 'Calendar'[Date].[Year] ) - 1 )
        RETURN
            IF (
                ISBLANK ( PreviousYearCount ),
                0,
                DIVIDE ( CurrentYearCount - PreviousYearCount, PreviousYearCount )
            )
    )
7.YoY Count indicator = SWITCH(
    TRUE(), [YoY Count] > 0,
    UNICHAR(9650), UNICHAR(9660))
