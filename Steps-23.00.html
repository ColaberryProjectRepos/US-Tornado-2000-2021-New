Add Tornado Impact Waterfall Chart 23

  • First create a measure description table.
  • On the Power BI Desktop Home ribbon, Click on Enter data, name the first column header as ID and the second column header as Measure and Name the table as US Tornado Impact. In the table, insert ID(1,2,3,4,5) and Measure(YoY Count,YoY Injury,YoY Fatality,YoY Width,YoY Length) respectively as shown below, then Click ok, Close & apply. 
  • Create a Measure called Variance in Accident Impact table to display the difference of accident measures on the Waterfall chart. Use the DAX formula below  
  • Variance = SWITCH(
  •   SELECTEDVALUE('US Tornado Impact'[Id]),
  •     1, [YoY Count],
  •     2, [YoY Injury],
  •     3, [YoY Fatality],
  •     4,[YoY Width],
  •     5,[YoY Length])
