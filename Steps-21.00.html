Adding Visual Charts-Year|Month|Day Line and Stack Column Chart 21


Steps
  • Click on an empty space on the canvas, and from the visualizations pane select Line and Stacked Column Chart
  •  Fill up figures provided below for the necessary fields.
  • Build Visual Tab:
  •  X-axis = Date,Year,Quarter,Month,Day
  •  Column y-axis = Count
  •   Line y-axis =YoY Count,YoY Fatality,YoY Injury,YoY Length,YoY Width
  •   Tooltips = Count,Fatality,Injury,Length,Width,YoY Count,YoY Fatality,YoY Injury,YoY Length,YoY Width
  • Cross-Report: On
Format Visual Tab:
General: Properties, Size:
  • Height=304
  •  Width=558
  •  Position:
  •  Horizontal=185
  •  Vertical=118
  •   Title=On, Text=Year|Month|Day, Font=Arial (12),Bold,Text color=#FFFFFF , Background color=#12239E, Horizontal alignment=Center
  •   Effects: Background=On, Background color=#E6E6E6, Transparency=20%,Â·      Visual boarder=ON, #E6E6E6. 
Visual:
  •   X-axis: Type=Categorical,Values: Font=Arial(9), Font color=#605E5C
  •  Title = Off      
  • Y-axis:Range:Minimum=0,Maximum=Auto,Values: Font=Arial(9), Font color=#605E5C, Display Units=Auto, Value Decimal places=Auto, Title = Off
  •   Secondary y-axis=On
  •   Font=Arial (9), Font color= #605E5C, Display Units=Auto, Value Decimal Places=Auto,Title = Off 
Legend=On, 
  • Position=Top left, Style = Marker(circle default), Match line color=On, Title=Off, Text=Segoe UI(8),Bold,Color=#605E5C
  • Gridlines: Horizontal=On, Style = Dotted,Color=#C8C6C4,Width=1 px,zoom slider=Off      
    • Column: Color= #70BBFF,Spacing:Inner padding=20 px,Minimum Category Width=20 px
  •  Lines: Line Style-Solid,Join type-Round,Stroke Width=3 px
  • Colors: YoY Width=   #744EC2 (Purple) ,YoY Length= #E044A7(Pink)  ,YoY Injury=      #007963(Green) ,YoY Fatality= #E66C37 ,YoY Count= #12239E(Blue)
  •  Shade area=On
  •  Markers= Off
  •  Data labels: Off
  •  Series labels: Off
